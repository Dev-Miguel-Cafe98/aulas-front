Exel e seus Conteitos

O Microsoft Excel é um software utilizado para elaboração, edição e gerenciamento de planilhas eletrônicas. Este software permite a criação e organização de dados a partir da criação e edição de planilhas onde os dados principais a serem trabalhados são números.

Principais funções do Excel Mas, o Excel é um programa tão completo que fica difícil escolher as principais funções! As cinco funções são: SOMA, MÉDIA, SE, CONT.SE E SOMA.SE. ... A função SOMA é a mais utilizada e serve para somar os valores de um intervalo de células.
